WooCommerce BingoPay Gateway.

== Description ==

Allow users to make payments with credit / debit cards via BingoPay.

== Frequently Asked Questions ==

= Is this PCI DSS compatible? =

Yes, absolutely. Visit BingoPay website for more info.

= Can I use this gateway for tests? =

Yes, you can use BingoPay's sandbox account and test credit cards numbers.
